Privacy Policy

Last Updated: January 15, 2026

This Privacy Policy describes how the Codex Templates API (“we,” “us,” or “the service”) collects, uses, and protects information when you access or use the API available at:

https://codex-templates.onrender.com

This policy is provided to meet transparency requirements for integrations, including Custom ChatGPT and other automated clients.

1. Information We Collect

We collect only the minimum information necessary to operate the API.

a. Session Information

When you interact with the API, we may create or reference a temporary session identifier (“session ID”) in order to:

  • Maintain continuity across API requests
  • Validate request context
  • Enforce rate limits and session expiration

Session data may include:

  • A generated session ID
  • Timestamps (session start, last access)
  • Temporary, non-personal session metadata

Session data does not contain personal identifying information by default.

b. API Request Data

We may temporarily process:

  • Request paths and parameters
  • Submitted payloads (such as template content sent for verification or upgrade)

This data is processed in-memory or transiently and is not used for profiling, tracking, or advertising.

2. Information We Do Not Collect

We do not intentionally collect or store:

  • Names, email addresses, or contact details
  • Government-issued identifiers
  • Payment or financial information
  • Location data beyond basic server logs
  • Behavioral analytics or tracking cookies

3. How Information Is Used

Information processed by the API is used solely to:

  • Fulfill API requests
  • Validate templates and registries
  • Maintain session integrity
  • Ensure system security and stability

We do not:

  • Sell data
  • Share data with third parties for marketing
  • Use data for advertising or profiling

4. Data Retention

  • Session data is temporary and expires automatically after a short time period.
  • Expired sessions are routinely cleaned up and discarded.
  • No long-term storage of request content is performed unless explicitly required by future features and documented accordingly.

5. Data Security

We take reasonable technical measures to protect the API and its data, including:

  • Session expiration and cleanup
  • Controlled access to API endpoints
  • Avoidance of unnecessary data persistence

However, no system can be guaranteed 100% secure, and use of the API is at your own risk.

6. Third-Party Services

The Codex Templates API may be hosted on third-party infrastructure providers. These providers may process limited technical data (such as IP addresses) as part of standard hosting and networking operations.

We are not responsible for the privacy practices of third-party platforms outside our direct control.

7. Children’s Privacy

The API is not intended for use by children under the age of 13, and we do not knowingly collect personal data from children.

8. Changes to This Policy

This Privacy Policy may be updated periodically. Updates will be reflected by the “Last Updated” date at the top of this document.

Continued use of the API after changes constitutes acceptance of the updated policy.

9. Contact

If you have questions about this Privacy Policy or the operation of the API, please contact the API administrator through the project’s official repository or support channel.
